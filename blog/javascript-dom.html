JS Fundamentals

1. What is JavaScript and what is its relationship to HTML and CSS

We can make a super basic webpage using HTML. We can add some style using CSS. But to make it truly functional, we need to add logic to it. This is where JavaScript comes in. It is a lightweight programming language that adds logic to the webpage.
To put it simply, if HTML is the frame of a car, and CSS is its body and paintjob, JavaScript is like the engine that would let us drive it around.
Adding JavaScript lets us control how the page should behave when the customer interacts with it.
